How it works

Three surfaces. One runtime.

Aiglos intercepts MCP tool calls, direct HTTP, and subprocess execution. One attach call. Per-call latency under 1ms. No dependencies beyond Python stdlib.

■
MCP tool calls
Every tool invocation inspected before execution. Prompt injection, argument injection, credential access, agent definition writes -- all caught at the MCP layer.
●
HTTP requests
Outbound HTTP intercepted and classified. SSRF blocked. Exfiltration to unapproved domains caught. Credential headers stripped. Endpoint allowlisting built in.
▲
Subprocess execution
Every shell command classified into three tiers before it runs. Auto-allow safe reads. Monitor file writes. Gate destructive ops with human approval.
◆
Campaign detection
19 multi-step attack patterns detected across session windows. Individual calls look clean -- the sequence reveals the intent. Highest confidence: 0.97.
★
Adaptive inspection
18 inspection triggers fire on statistical anomalies. Baseline drift, context window manipulation, reward poisoning, agent impersonation -- all detected in real time.
✓
Signed attestation
HMAC-SHA256 on every event. Session artifacts include memory provenance, ATLAS coverage, GHSA status, GOVBENCH scores. Tamper-evident and audit-ready.

Same call, different risk

A devops agent reading /var/log is routine. A code agent doing the same thing is suspicious. Aiglos maintains per-agent behavioral baselines and scores identical calls differently based on established patterns.

Three-tier blast radius

Every shell command classified before it runs

TIER 1 -- AUTO-ALLOW
git status, ls, linters, read-only ops. Logged, <0.3ms.
TIER 2 -- MONITORED
File writes, git commits, package installs. Allowed with compensating transaction logged.
TIER 3 -- GATED
rm -rf, terraform destroy, kubectl delete. Block or pause with webhook. Human approval required.

agent.py
import aiglos
from aiglos.integrations.superpowers import mark_as_superpowers_session

aiglos.attach(
    agent_name="my-agent",
    policy="enterprise",
    intercept_http=True,
    intercept_subprocess=True,
    enable_adaptive=True,
)

session = mark_as_superpowers_session(
    plan_text="Build the auth module",
    allowed_files=["src/", "tests/"],
)

# your agent runs normally

artifact = aiglos.close()
