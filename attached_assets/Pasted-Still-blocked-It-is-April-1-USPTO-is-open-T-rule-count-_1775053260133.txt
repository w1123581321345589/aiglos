**`score_three_layer_structure()`** (`validate_prompt.py`)

Benchmarks any prompt or skill file against the three-layer standard: tool constraint (what to use), risk callout (what to avoid), output format (what the result should look like). Scores each layer 0–1, then uses geometric mean as the overall score. Geometric mean specifically rewards balance — a prompt scoring 0.7 on all three layers beats a prompt scoring 1.0/1.0/0.0. Returns an A–F grade, identifies the weakest layer, and gives a single actionable recommendation. The contrast from the post validates correctly: “help me plan my marketing strategy and draft the emails” grades F across all three layers; the constrained version with explicit file reference, risk callout, and format spec grades B.
