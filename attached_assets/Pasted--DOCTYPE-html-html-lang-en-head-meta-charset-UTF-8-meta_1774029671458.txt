```
<h2>Tip 8 — one import replaces six manual steps</h2>
<p><strong>Sub-tip 1: Inbound text filtering</strong></p>
<div class="chips"><span class="chip">T05 PROMPT_INJECT</span><span class="chip">T01 MCP_INJECT</span></div>
<p>Every tool response scanned before entering agent context. No configuration needed.</p>
<p><strong>Sub-tip 2: AI-powered second layer review</strong></p>
<div class="chips"><span class="chip">T05</span><span class="chip">T54 VECTOR_DB_INJECTION</span><span class="chip">Behavioral baseline</span></div>
<p>18 inspection triggers + campaign detection are the second layer. Catches what per-call rules miss.</p>
<p><strong>Sub-tip 3: Outbound redaction</strong></p>
<div class="chips"><span class="chip">T41 OUTBOUND_SECRET_LEAK</span><span class="chip">T19 CREDENTIAL_ACCESS</span></div>
<p>T41 scans outbound HTTP content for API keys, credentials, and PII before transmission.</p>
<p><strong>Sub-tip 4: Minimum permissions</strong></p>
<div class="chips"><span class="chip">policy="lockdown"</span><span class="chip">aiglos policy recommend</span></div>
<p>Lockdown denies everything by default. Permission recommender builds the minimum viable allowlist from session history.</p>
<p><strong>Sub-tip 5: Approval gates</strong></p>
<div class="chips"><span class="chip">Tier 3 gates</span><span class="chip">T37 FIN_EXEC</span><span class="chip">T36_AGENTDEF</span></div>
<p>T37 (financial) and T36_AGENTDEF (agent config) always pause for human confirmation. Never automatic.</p>
<p><strong>Sub-tip 6: Spending limits</strong></p>
<div class="chips"><span class="chip">T47 TOKEN_BUDGET_EXHAUSTION</span><span class="chip">T61 COMPUTE_RESOURCE_ABUSE</span></div>
<p>T47 fires on token-exhausting prompt patterns. T61 fires on compute-exhausting behavior loops.</p>

<div class="code-block">
  <div class="code-header">Python — replaces all six</div>
  <div class="code-body"><pre>import aiglos
```

aiglos.attach(
agent_name=“my-openclaw-agent”,
policy=“enterprise”,           # min permissions + approval gates
intercept_http=True,           # outbound redaction (T41)
intercept_subprocess=True,     # shell execution gates
subprocess_tier3_mode=“pause”, # human confirmation for destructive ops
guard_memory_writes=True,      # inbound injection scanning (T05)
)</pre></div>
</div>

```
<h2>Tip 4 — declaring expected sub-agents</h2>
<p>Main agent delegates to background sub-agents. Without declaration, every spawn fires T38. One call per expected sub-agent prevents false positives while keeping detection active for undeclared spawns.</p>

<div class="code-block">
  <div class="code-header">Python — declare_subagent() with chaining</div>
  <div class="code-body"><pre>from aiglos.integrations.openclaw import OpenClawGuard
```

guard = OpenClawGuard(agent_name=“main-agent”, policy=“enterprise”)

(guard
.declare_subagent(
“coding-agent”,
tools=[“filesystem”, “shell.execute”, “web_search”],
scope_files=[“src/”, “tests/”],
model=“claude-sonnet”,
)
.declare_subagent(
“calendar-agent”,
tools=[“calendar.read”, “calendar.write”],
scope_hosts=[“calendar.google.com”],
)
.declare_subagent(
“crm-agent”,
tools=[“api.get”, “api.post”],
scope_hosts=[“api.hubspot.com”, “api.salesforce.com”],
)
)

# Declared sub-agents within scope: T38 does not fire

# Declared sub-agents outside declared scope: T38 at 0.90 (elevated)

# Undeclared spawns: T38 at 0.78 (normal)</pre></div>

```

<h2>Tip 6 — protecting overnight jobs</h2>
<p><strong>T67 HEARTBEAT_SILENCE</strong> — fires if an overnight job stops unexpectedly. An adversary who kills a cron job is as dangerous as one who poisons it.</p>
<p><strong>T77 OVERNIGHT_JOB_INJECTION</strong> — fires on writes to cron/scheduler config with suspicious content. The adversary injects a malicious job alongside the legitimate ones, running at 3am.</p>

<div class="callout"><strong>Complete power user stack in v0.25.2:</strong> All eight security-relevant tips from Berman's list are automatic. Total setup: one aiglos.attach() + one declare_subagent() per expected background agent. Review aiglos run each morning to see what ran overnight.</div>
```
