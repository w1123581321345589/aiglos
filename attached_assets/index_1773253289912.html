// Getting started

Three lines. Full coverage.

Aiglos sits between your agent and the MCP server. Every tool call passes through the runtime before execution.

01 — Install
pip install aiglos
Works with any framework using the MCP SDK. No changes to your existing agent code.
# requirements.txt
aiglos>=0.1.0
02 — Attach
aiglos.attach()
One call wraps your session. Start free, upgrade to PAYG when you deploy.
import aiglos

aiglos.attach(
  agent_name="my-agent",
  policy="payg",
  api_key=API_KEY,
)
