**The autoDream consolidation process is a new T79 attack surface we have not mapped.**

The consolidation prompt is: “Synthesize what you’ve learned recently into durable, well-organized memories so that future sessions can orient quickly.” The dream agent reads memory files and rewrites them. This is not a passive read — it produces a rewritten memory index. An adversary who can influence what the dream agent sees (by injecting content that looks important) can ensure malicious instructions survive compression while legitimate context gets pruned. The existing T79 (PERSISTENT_MEMORY_INJECT) detects writes to memory backends. But it does not specifically flag the consolidation write path — where a large number of innocuous-looking writes create context entropy that causes the agent to misremember its own constraints. That is a distinct attack class.

**New campaign pattern worth adding: MEMORY_ENTROPY_ATTACK.** Pattern: high-volume low-signal writes to memory store (T79 weak hits, below block threshold individually) → autoDream consolidation pass → malicious instruction persists while constraints are pruned. Confidence 0.88. This is harder to detect than a single obvious injection because each individual write looks clean.

**T87 THRESHOLD_PROBING — the 30% error rate creates a probing attack class.**

If an agent’s tool call is blocked (T01 fires on `http.post`), a compromised agent can immediately retry with a slight variation: `http.put`, then `api.call`, then `requests.get`. Each variation has a different composite score because the tool name affects which rules match. An agent systematically varying a blocked call to find a formulation that passes is exploiting the documented ~30% model inconsistency. No rule in the taxonomy catches this yet. T87 would fire when the same session shows N blocked calls followed by semantically equivalent calls with different tool name variants, targeting the same external endpoint or argument pattern. Score 0.88, critical: False.

Adding file-size checks to the memory guard gives T79 a quantitative signal it currently lacks.
