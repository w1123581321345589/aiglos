      <div class="code-body"><pre><span style="color:#4ade80">$</span> aiglos agents scaffold

<strong>aiglos agents scaffold</strong>

Describe each agent in plain English. Press Enter when done.

Agent 1: CEO agent using Opus for decisions, push rights to main branch
Agent 2: Assistant agent using Kimi for research, side branches only
Agent 3:

<span style="color:#6e7681"># Generated by: aiglos agents scaffold</span>
