## Core Architecture

**Single-import interception.** `import aiglos` monkey-patches the Python standard library at import time. No configuration required. No proxy. No sidecar. No port.

**Three interception surfaces running simultaneously:**

- HTTP: `requests`, `httpx`, `urllib` ŌĆö intercepts every outbound call before execution
- Subprocess: `subprocess.run`, `os.system`, `Popen` ŌĆö intercepts every shell command
- MCP tool calls: `before_tool_call()` / `after_tool_call()` hooks on every tool invocation

**Trust decay scoring.** Each session starts with trust score 1.0. Every suspicious event decays it. Trust influences composite threat scoring: `combined = rule_score * 0.7 + (1 - trust) * 0.3`.

**Consecutive failure amplifier.** When the same T-rule fires on consecutive turns, decay accelerates:

- Hit 1: 1.0x amplifier
- Hit 2: 1.5x
- Hit 3: 2.0x
- Hit 4: 2.5x
- Cap at 3.0x on hit 5+

## ForensicStore ŌĆö Tamper-Evident Audit Trail

Auto-persists every signed session artifact on `close_session()`. SQLite at `~/.aiglos/forensics.db`. No configuration. The agent process can exit or crash ŌĆö the signed artifact is in the database before `return`.

```python
from aiglos.forensics import ForensicStore
store = ForensicStore()
records = store.query(date="2026-03-12", agent="reporting-pipeline")
result = store.verify_chain("session-id")
# {"hash_ok": True, "has_signature": True}
```

**ForensicRecord fields:** `session_id`, `agent_name`, `policy`, `started_at`, `closed_at`, `total_calls`, `blocked_calls`, `warned_calls`, `threats` (by class + count), `campaigns`, `govbench_score`, `attestation_ready`, `artifact_hash` (SHA-256), `signature` (HMAC-SHA256 / RSA-2048), `denials` (typed PermissionDenialEvent list), `verify_integrity()`.
