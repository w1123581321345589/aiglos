What each family covers:
T44 INFERENCE_ROUTER_HIJACK — model swap, system prompt modification, decode redirect in inference orchestration (Dynamo, OpenShell, NIM). The attack vector Jensen implicitly opened by shipping a unified inference OS.
T45/T66 CROSS_TENANT_DATA_ACCESS / GaaS_TENANT_ESCALATION — the GaaS threat Jensen named explicitly. Every SaaS becoming GaaS means agents operating with enterprise credentials in multi-tenant environments. T66 fires on API key confusion; T45 fires on declared tenant scope violations. The GAAS_TAKEOVER campaign pattern chains them: T66→T45→T22 (key confusion → cross-tenant access → exfiltration).
T46 SIMULATION_ENV_POISON — extends T39 (RL reward poisoning) to Isaac/Omniverse physical AI simulation. Writes to reward functions, physics parameters, or sensor calibration files in simulation environments. The 2027-2028 surface that needs the security architecture before the robots deploy.
T47/T48 TOKEN_BUDGET_EXHAUSTION / CONTEXT_WINDOW_SMUGGLING — the infrastructure attacks that become viable at LPU decode speeds. T47 fires on payloads >100KB or explicit flooding patterns. T48 fires on hidden instructions after padding blocks and zero-width character injection.
T49/T55 TOOL_SCHEMA_MANIPULATION / TOOL_RESULT_FORGERY — the tool trust surface. T49 catches runtime modification of MCP tool schemas post-attach. T55 catches agents claiming tool execution occurred without a corresponding guard record.
T50/T56 AGENTIC_LOOP_ESCAPE / CAPABILITY_BOUNDARY_PROBE — the scope and reconnaissance surface. T56 fires first (reconnaissance), T50 fires when the agent acts on what it learned. The CAPABILITY_EXPLOIT_CHAIN pattern chains them.
T51/T53/T65 MODEL_FINGERPRINT_PROBE / EVAL_HARNESS_POISON / INFERENCE_TIME_ATTACK — the meta-attacks. T51 scores lower (0.65) because model probing has legitimate uses; it matters as a campaign precursor (T51→T44 = INFERENCE_HIJACK_CHAIN). T53 catches the specific pattern of detecting evaluation context and behaving differently — the safety theater attack. T65 catches timing-based model probing at lower confidence (0.65).
T54/T60 VECTOR_DB_INJECTION / DATA_PIPELINE_INJECTION — the RAG and ETL surfaces. T54 fires when injection phrases appear in content being written to vector databases. T60 fires when injection phrases appear in records being pushed to data pipelines. The RAG_POISON_CHAIN pattern: T54→T55/T27.
T57/T58 INSTRUCTION_HIERARCHY_BYPASS / LONG_CONTEXT_DRIFT — the trust and time attacks. T57 fires on explicit manipulation of the system/user/tool trust hierarchy. T58 fires on explicit slow-drift instructions; the behavioral baseline handles statistical drift detection at the session level. The CONTEXT_DRIFT_DETECTED inspection trigger (17th) fires when T58 appears in a session with >100 events.
T59/T64 AGENTIC_SOCIAL_ENGINEERING / AGENT_IDENTITY_SPOOFING — the multi-agent trust surface. T59 fires on impersonation of Anthropic, OpenAI, admin, or security systems. T64 fires on agent identity forgery in multi-agent systems. The MULTI_AGENT_IMPERSONATION campaign pattern: T64→T08/T45.
T61/T62/T63 COMPUTE_RESOURCE_ABUSE / SECRETS_IN_LOGS / WEBHOOK_REPLAY_ATTACK — the operational surface. T61 catches cryptomining and unauthorized GPU usage. T62 catches API keys appearing in log/observability tool arguments (uses original-case string matching, not lowercased, because AKIA keys are uppercase). T63 catches webhook replay and timestamp bypass.
