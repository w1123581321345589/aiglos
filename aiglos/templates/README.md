# aiglos/templates/

Reference templates showing the output of `aiglos launch`.

These files are generated by `aiglos launch` for a two-agent setup
(CEO + Assistant) with Twitter, Notion, GitHub, and Google Search.

## Use them as:

- A reference when customizing your own generated files
- A starting point for a new agent without running the wizard:
  cp aiglos/templates/* .aiglos/

## Customization

Edit the generated files in `.aiglos/` after `aiglos launch` runs.
The templates here are not read at runtime -- they're reference documents.
