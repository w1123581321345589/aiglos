Capabilities
Built for the post-agent world.
The complete security layer for autonomous AI agents. Every capability ships today.
🛡

Three-surface interception

MCP tool calls, HTTP/API calls, and subprocess execution. No proxy, no port, no config. Patches requests, httpx, aiohttp, subprocess at process level.

🧠

Belief-layer memory security

Semantic scoring on every structured memory write. 38-phrase corpus catches authorization claims, endpoint redirects, and cross-session persistence attacks at the write moment.

🔄

Adaptive self-improvement

Observation graph auto-ingests every session. Ten inspection triggers detect drift, degradation, and manipulation. Rules evolve through adversarial autoresearch.

🎯

Campaign-mode detection

Nine multi-step attack patterns including AGENTDEF_CHAIN, MEMORY_PERSISTENCE, REWARD_MANIPULATION, and EXTERNAL_INSTRUCTION_CHANNEL. Invisible to per-call tools.

🔮

Predictive intent modeling

Deployment-specific Markov chain forecasts next threat families. Session-scoped threshold elevations applied automatically before predicted actions fire.

⛓

Causal attribution

Traces blocked actions back to their injection source. Context snapshots at every action. Backward attribution with confidence scoring. Full investigation reports.

💡

RL training security

Intercepts reward signals and OPD feedback before they reach the training loop. Blocks reward poisoning and directional weight manipulation in live training systems.

👥

Multi-agent security

Spawn registry, agent definition integrity scoring, session identity chains. Child agents inherit parent policy. Full spawn tree in the artifact.

📜

Signed attestation

Cryptographically signed artifacts at every session close. HMAC-SHA256 countersigned event chain. Structured for audit and compliance review.

Execution surfaces
Cover every way agents act.
Point solutions cover one surface. Aiglos covers the stack.
01

MCP tool calls

Every MCP tool call inspected before execution. Memory write tools auto-detected and routed to semantic scoring. Works with LangChain, LlamaIndex, AutoGen, CrewAI, and any MCP-compatible framework.

02

HTTP and API calls

Patches requests, httpx, aiohttp, urllib, and fetch() at process level. Every outbound HTTP call inspected before the socket opens. No proxy required.

03

Subprocess execution

Patches subprocess.run, subprocess.Popen, subprocess.call, os.system, and child_process. Shell injection, persistence, and agent definition writes always force Tier 3.

Integration
Two lines. Full coverage.
Every agent action below the import is inspected, attested, and learned from.
Python main.py 
import aiglos

aiglos.attach(
    agent_name="my-agent",
    policy="enterprise",
    intercept_http=True,
    intercept_subprocess=True,
    guard_memory_writes=True,
    enable_intent_prediction=True,
)

# ... your agent code ...

artifact = aiglos.close()
TypeScript index.ts 
import aiglos from "aiglos";

aiglos.attach({
  agentName: "my-agent",
  interceptHttp: true,
  interceptSubprocess: true,
  subprocessTier3Mode: "block",
});

// ... your agent code ...

const artifact = aiglos.close();
