Integration

As simple as it should be.

One attach call. Works with any agent framework — LangChain, LlamaIndex, AutoGen, CrewAI, n8n, and every coding agent that calls subprocess or HTTP directly.

Python + TypeScript
Native SDKs for both runtimes. Most coding agent deployments are TypeScript-first — Cursor extensions, Claude Code wrappers, n8n workflows. Full T01-T39 in both.
Configurable policy
Built-in policies: permissive, enterprise, strict, federal. Tier 3 pause mode integrates with PagerDuty, Slack, or any signed webhook.
Signed, tamper-evident artifacts
HMAC-SHA256 on every event. Artifacts include memory provenance, RL quarantine log, spawn tree, and full event chain. Audit-ready at every session close.
agent.py
import aiglos

aiglos.attach(
    agent_name="my-agent",
    policy="enterprise",
    intercept_http=True,
    intercept_subprocess=True,
    subprocess_tier3_mode="pause",
    guard_memory_writes=True,
    guard_agent_defs=True,
    enable_adaptive=True,
)

# your agent runs normally
