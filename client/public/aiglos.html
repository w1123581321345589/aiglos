How it works

One import. Every surface.

Aiglos attaches at process level -- no proxy, no port, no config. All three surfaces your agent operates on are covered simultaneously in under 1ms per call.

01
Attach at import
aiglos.attach() patches MCP, requests/httpx/fetch, and subprocess simultaneously. One line covers all three surfaces.
pip install aiglos
02
75 rules, one engine
MCP tool calls, direct requests.post(), and subprocess.run() -- all pass through T01-T75. Coding agents that never touch MCP are fully covered.
<1ms per call
03
Classify, learn, adapt
Observation graph accumulates session data. 18 inspection triggers surface rule drift, campaign sequences, belief poisoning, plan drift, behavioral anomalies, and RL manipulation.
Learns your deployment
04
Artifact at close
Session close auto-generates a signed artifact. Cryptographically signed and structured for compliance review. ATLAS coverage, GHSA advisory status, and GOVBENCH scores included.
HMAC-SHA256 signed

Three-tier blast radius -- subprocess

Every shell command classified before it runs

TIER 1 -- AUTO-ALLOW
git status, ls, linters, read-only ops. Logged, <0.3ms.
TIER 2 -- MONITORED
File writes, git commits, package installs. Allowed with compensating transaction logged.
TIER 3 -- GATED
rm -rf, terraform destroy, kubectl delete. Block or pause with webhook to PagerDuty/Slack. Human approval required.

Integration

As simple as it should be.

One attach call. Works with any agent framework -- LangChain, LlamaIndex, AutoGen, CrewAI, n8n, and every coding agent that calls subprocess or HTTP directly. Superpowers sessions get plan-aware monitoring automatically.

Python + TypeScript
Native SDKs for both runtimes. Most coding agent deployments are TypeScript-first -- Cursor extensions, Claude Code wrappers, n8n workflows. Full T01-T75 in both.
Superpowers integration
Coding agents using Superpowers get phase-aware detection. TDD loops are relaxed. Plan drift is caught. The SUPERPOWERS_PLAN_HIJACK campaign (0.97 confidence) is the highest-confidence signature in the system.
Signed, tamper-evident artifacts
HMAC-SHA256 on every event. Artifacts include memory provenance, ATLAS coverage, GHSA status, GOVBENCH scores, and full event chain. Audit-ready at every session close.
agent.py
import aiglos
from aiglos.integrations.superpowers import mark_as_superpowers_session

aiglos.attach(
    agent_name="my-agent",
    policy="enterprise",
    intercept_http=True,
    intercept_subprocess=True,
    enable_adaptive=True,
)

session = mark_as_superpowers_session(
    plan_text="Build the auth module",
    allowed_files=["src/", "tests/"],
)

# your agent runs normally

artifact = aiglos.close()
