Common questions

Aiglos intercepts every MCP tool call your AI agents make — before it executes. One import, no proxy server, no config. It patches into the MCP client at runtime and runs every call through 36 rule families in under a millisecond. Malicious calls get blocked. Suspicious ones get flagged. Everything else runs normally.
Claude Code asks for your approval before running commands. Aiglos inspects what those commands actually are against a continuously updated threat database — prompt injection patterns, credential extraction attempts, supply chain attacks, SSRF, shell abuse — before you ever see the approval prompt. It’s the layer between the model and your machine.
