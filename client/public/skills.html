How it works

Static badges + runtime detection

ClawKeeper scans skill packages for known patterns. Aiglos goes further: it watches what skills actually do at runtime and detects behavioral anomalies in real time.

Static analysis

ClawKeeper badge scanning

ClawKeeper scans 13,000+ ClawHub skills and assigns each a security badge based on static analysis of package contents, dependencies, and known threat signatures.

  • Package manifest and dependency analysis
  • Known malicious pattern matching
  • Obfuscation and suspicious code detection
  • Badge assignment: Clean, Suspicious, or Blocked
  • Community-sourced reporting pipeline
Runtime detection

Aiglos behavioral analysis

Aiglos monitors skill behavior at runtime. A skill that looks clean in static analysis can still exfiltrate data, inject prompts, or escalate privileges during execution.

  • Real-time tool call interception (T01-T43)
  • Behavioral anomaly scoring per session
  • Campaign pattern detection across deployments
  • Federation intelligence from the Aiglos network
  • Automated T30 SUPPLY_CHAIN rule enforcement
