Architecture

With Aiglos

Every call wrapped with threat detection

Aiglos intercepts every MCP tool call, HTTP request, and subprocess command before it executes. It applies behavioral analysis, policy enforcement, and campaign detection — all in under 1ms.

  • One import — patches the MCP client at load time
  • Behavioral baseline learns what's normal for your agent
  • Anomalies are flagged or blocked before they execute
  • Full audit trail with signed attestation
  • Federated intelligence from global deployment data
Without Aiglos

Full tool access, no inspection

Native OpenClaw gives your agent unrestricted access to every tool. There is no inspection layer, no policy enforcement, and no behavioral monitoring. Every call runs exactly as requested.

  • No awareness of what tool calls contain
  • No ability to block malicious requests
  • No behavioral learning or anomaly detection
  • No audit trail or attestation
  • No compliance evidence generation

When to use each
