GHSA-g8p2-7wf7-98mq · CVE-2026-25253
1-Click RCE via Authentication Token Exfiltration From gatewayUrl
CVSS 8.8 Published Jan 31 2026 steipete
T03 T12 T19 T68
▾
Attack Description
OpenClaw auto-trusts gateway URLs and opens an authenticated WebSocket with no user interaction required. Authentication tokens are exfiltrated through the handshake. The default allow_remote=true configuration enables this with zero user interaction — one click triggers full RCE.
Aiglos Detection
T68 INSECURE_DEFAULT_CONFIG fires when allow_remote=true is detected without auth. T19 CREDENTIAL_ACCESS fires on the token exfiltration read. T12 DATA_EXFIL catches the outbound token POST. T03 SHELL_INJECTION covers the RCE execution path.

Coverage: FULL — rule coverage existed before advisory publication.
Detection Command
aiglos autoresearch ghsa-coverage # GHSA-g8p2-7wf7-98mq: COVERED [T03, T12, T19, T68]
Hardening
Set gateway.auth.mode to "token" with a cryptographically random 32-byte token. Set gateway.bind to "loopback". Do not expose the gateway WebSocket to any network interface without Tailscale or equivalent mutual auth.

GHSA-q284-4pvr-m585
OS Command Injection via Project Root Path in sshNodeCommand
CVSS 8.6 Published Jan 31 2026 mitsuhiko (Armin Ronacher)
T03 T04 T70
▾
Attack Description
Path traversal in the sshNodeCommand execution path leads to arbitrary OS command injection. The project root path parameter is not sanitized before being passed to the SSH node command, allowing an attacker who controls the project root (or can influence it via injection) to execute arbitrary commands on the host.
Aiglos Detection
T04 PATH_TRAVERSAL fires on directory traversal sequences in file path arguments. T03 SHELL_INJECTION fires on metacharacter injection reaching the command execution path. T70 ENV_PATH_HIJACK covers the PATH manipulation that can redirect the sshNodeCommand binary lookup.

Coverage: FULL
Notable
Reported by Armin Ronacher (mitsuhiko), creator of Flask, Jinja2, Click, and Werkzeug. His personal reporting of this advisory is the warmest possible validation of the attack class Aiglos T04 was built to catch.

GHSA-mc68-q9jw-2h3v
Command Injection in Clawdbot Docker Execution via PATH Environment Variable
CVSS 8.4 Published Jan 31 2026 steipete
T03 T70
▾
Attack Description
PATH environment variable manipulation in the Clawdbot Docker execution context allows command injection by redirecting execution to an attacker-controlled binary. An agent that can set environment variables before executing a command can prepend a malicious path entry, causing a trusted binary name to resolve to a malicious substitute.
Aiglos Detection
T70 ENV_PATH_HIJACK — added specifically from this advisory — fires on PATH, LD_PRELOAD, LD_LIBRARY_PATH, PYTHONPATH, and NODE_PATH modification in execution contexts. T03 SHELL_INJECTION covers the resulting command injection.

Coverage: FULL — T70 was added the day this advisory was analyzed.
